Esta é a interface de autenticação do **Controle de Gastos**, um sistema focado no gerenciamento de finanças pessoais. A página de login conta com validações dinâmicas em tempo real para garantir uma melhor experiência e segurança do usuário.

## 🚀 Funcionalidades

- **Validação de E-mail:** Verifica se o campo está vazio ou se o formato do e-mail é inválido.
- **Validação de Senha:** Verifica o preenchimento do campo de senha.
- **Habilitação Dinâmica:** Os botões "Entrar" e "Recuperar Senha" são ativados dinamicamente apenas quando os critérios de validação são atendidos.
- **Interface Responsiva:** Adaptável para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

O front-end foi construído utilizando tecnologias web padrão, sem a necessidade de frameworks externos:

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização moderna e layout responsivo.
- **JavaScript (Vanilla):** Lógica de validação e manipulação do DOM.
